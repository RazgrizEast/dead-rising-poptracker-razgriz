# dead-rising-poptracker-razgriz
A poptracker for Dead Rising Deluxe Remaster on Archipelago

I am not an expert, so don't expect anything fancy. I did the best I could with the limited skills I have.

There are probably logic rules I misunderstood or randomization option combinations that I messed up on.

You can ping me on Discord (I am in the Archipelago After Dark server as Razgriz) to let me know of any errors.

TO USE: Simply download the .zip file and put it in your Poptracker packs folder. It should work then.

IMPORTANT NOTE: Due to Dead Rising Deluxe Remaster hogging computer resources, the PopTracker will not automatically update if DRDR is played in fullscreen. Even if you have the PopTracker open on a second monitor, it will only update when you minimize Dead Rising (and even then it will have visual glitches in the item grid). If you want to use the PopTracker at the same time as you play, you need to set DRDR to either Windowed or Borderless Windowed. This is a limitation between DRDR and PopTracker and I do not think I can fix it. 

KNOWN LIMITATIONS: I do not yet know how to add offline settings to a PopTracker. When you load it, it will assume all settings are normal (regular game, no restricted items). However, when you connect it to an Archipelago server, it will access your slot data and adjust accordingly.
